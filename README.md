🧠 Kraepelin Cognitive Analytics (AI-Based)

Live Preview: [https://cerminsiji.github.io/Kraepelin-Test/](https://cerminsiji.github.io/Kraepelin-Test/)

---

📌 Overview

Kraepelin Cognitive Analytics adalah aplikasi berbasis web untuk simulasi dan analisis Tes Kraepelin yang ditingkatkan dengan:

- 📊 Statistik performa
- 🤖 Machine Learning (TensorFlow.js)
- 🧠 Analisis psikologis mendalam
- 📱 Desain mobile-friendly

Aplikasi ini tidak hanya menghitung benar/salah, tetapi juga menganalisis pola kerja mental, konsistensi, dan ketahanan kognitif.

---

🚀 Features

🧠 Psychological Analysis

- Work Speed
- Accuracy Control
- Consistency
- Mental Endurance
- Impulse Control
- Focus Stability
- Stress Resistance

🤖 Machine Learning

- Feature extraction dari data performa
- Neural network (TensorFlow.js)
- Automatic classification (profil pengguna)

📊 Visualization

- Line chart performa per kolom
- Radar chart profil psikologis

📱 Mobile Friendly

- Responsive UI
- Optimized untuk smartphone

---

🧪 How It Works

1. User menyelesaikan tes Kraepelin
2. Sistem mengumpulkan data per kolom:
   - jumlah benar
   - jumlah salah
   - waktu
3. Feature engineering:
   - mean speed
   - accuracy
   - variability
   - trend
   - fatigue
4. Machine Learning memproses data
5. Sistem menghasilkan:
   - Profil psikologis
   - Insight otomatis
   - Visualisasi performa

---

🧮 Metrics Calculation

Metric| Description
Speed| Rata-rata jawaban benar
Accuracy| Persentase jawaban benar
Consistency| Stabilitas performa

Fatigue| Indikasi kelelahan
