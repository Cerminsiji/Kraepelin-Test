## 📌 Overview

**Kraepelin Test Pro** adalah aplikasi berbasis web yang mensimulasikan **Tes Kraepelin (Pauli Test)** — metode psikotes klasik yang digunakan dalam **HR assessment** untuk mengukur performa kognitif seseorang.

### 🎯 Tujuan Pengukuran:
- ⚡ Kecepatan kerja (Work Speed)
- 🎯 Ketelitian (Accuracy)
- 📈 Konsistensi (Stability)
- 🧠 Daya tahan mental (Endurance / Fatigue)
- ⚖️ Kontrol impuls (Impulse Control)

Aplikasi ini dirancang dengan pendekatan:
- Cepat
- Minimalis
- Responsif
- Fokus pada performa real-time

---

## ⚙️ How It Works

Pengguna akan diberikan operasi penjumlahan sederhana secara terus-menerus.

### 📊 Contoh:

9 + 6 = 15 → Jawaban: 5 
8 + 7 = 15 → Jawaban: 5

### 📏 Aturan:
- Angka selalu antara **0 – 9**
- Hasil penjumlahan bisa lebih dari 9
- Jawaban adalah **angka satuan (%10)**
- Tekan **ENTER** untuk lanjut ke soal berikutnya
- Tes berjalan secara **kontinu tanpa jeda**

---

## ✨ Key Features

### 🧪 Core Test Engine
- Continuous addition (stream-based)
- Real Kraepelin logic (bukan per kolom statis)
- Keyboard-first interaction (super cepat)

### ⚡ Performance Tracking
- Total answered questions (Speed)
- Accuracy rate (% benar)
- Reaction time (ms)
- Fatigue analysis (awal vs akhir)

### 💾 Persistence System
- Auto-save menggunakan **localStorage**
- Resume test setelah refresh
- Aman dari gangguan koneksi

### 📱 User Experience
- Mobile-friendly
- UI minimalis (tanpa distraksi)
- Input instan (tanpa tombol klik)

### 🧠 Psychological Insight
- High Performer
- Stable Performer
- Impulsive
- Easily Fatigued

### 📊 Metrics & Analysis

***Metric	Deskripsi***
* Speed	: Jumlah soal yang dikerjakan
* Accuracy :	Persentase jawaban benar
* Reaction : Time	Waktu rata-rata per soal
* Fatigue : Index	Penurunan performa
* Consistency	: Stabilitas performa
